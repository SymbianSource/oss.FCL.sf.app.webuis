This directory contains several scripts to generate a Browser SISx package

Currently, the associated scripts have been tested building SISx packages only 
for the S60 3.2.3 platform.

to create SISx package for Browser 7.1, use buildIADsis.pl.

Known issues 
* Favengine server needs to be rebooted to take effect

Future release support:
The following features will be supported soon 
* Support for S60 5250 platform
* Support UREL and UDEB version binaries for both winscw and armv5
